Follow these instructions to get started with the WavePlot system on Microsoft Windows operating systems.

Windows Windows 7 or later

  1. To begin with, you will need to download and install the WavePlot Client Library (WCL), which is a piece of software which provides the backbone for all other WavePlot programs. Pre-built binaries of the WCL are available below, along with the source code.

    libwaveplot-0.exe (recommended) x64/x86 9.4 MB md5
    libwaveplot-0.0.tar.gz - 0.7 MB md5

  2. Once the libraries are installed, you can download the waveplot-scanner Python 2.7 script and begin scanning your music collection. An exectable installer for the latest released version of the script is available below. You can also obtain it in the 'waveplot-scanner' package from PyPI, using pip.

    waveplot-scanner-0.exe x64/x86 5.8MB md5

  3. After downloading the script, you can run it as follows, on any directory containing music files. Register here to get an editor key.

                waveplot-scanner.exe <editor_key_here> <path\to\music\here>
