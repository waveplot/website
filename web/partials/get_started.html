Get Started!

There are two ways to obtain WavePlot for Windows.

Binaries

Firstly, you can download an installer which will install the WavePlot Imager and the scanning script into a directory of your choosing. This is available here. Please note, you'll need to install Python 2.7 and the Python packages mutagen and Requests to use the provided scanner script.

Source Code

The second way is to compile the source code yourself. You can download the source files from GitHub using this link. You'll need to install MinGW and libav 0.8.7 to compile on Windows, and once again, you'll need Python 2.7 and the packages mutagen and Requests to run the scanning script.

Using

Once you've installed WavePlot, navigate to the directory with the files you want to analyse for WavePlots, and open a command prompt or terminal. Then, in the command prompt or terminal, type "<path_to_scanner>/scanner.py", replacing <path_to_scanner> with the directory containing scanner.py and WavePlotImager

At this point, email me if it all goes wrong. :)

To get started with WavePlot, you'll need to download a few files:

  • The WavePlot Imager
    • Ubuntu x64 Binary (for 64 bit systems)
    • Source Code with Makefile for UNIX-based systems
    • libavcodec, libavformat and libavutil development packages
  • The WavePlot Scanner
    • You'll first need to install Python 2.7.
    • Then you'll need to download and install the "mutagen" and "requests" python packages.

Once you have both of the WavePlot downloads and their dependencies, place WavePlotImager and scanner.py in a folder. Navigate to the directory with the files you want to analyse for WavePlots, and open a command prompt or terminal. Then, in the command prompt or terminal, type "<path_to_scanner>/scanner.py", replacing <path_to_scanner> with the directory containing scanner.py and WavePlotImager
