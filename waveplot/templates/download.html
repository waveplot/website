To get started with WavePlot, you'll need to download a few files:

  • The WavePlot Imager
    • Windows x86 Binary (for 32 or 64 bit systems)
    • Ubuntu x64 Binary (for 64 bit systems)
    • Source Code with Makefile for UNIX-based systems
    • Source Code with Makefile for MinGW32 on Windows
      • You'll need to download and install libavcodec, libavformat, libavutils (http://ffmpeg.org/download.html) and libpng to compile from source.
    • The WavePlot Scanner
      • You'll first need to install Python 2.7.
      • Then you'll need to download and install mutagen, using the included installation instructions.

    Once you have both of the WavePlot downloads and their dependencies, place WavePlotImager(.exe) and scanner.py in a folder. Navigate to the directory with the files you want to analyse for WavePlots, and open a command prompt or terminal. Then, in the command prompt or terminal, type "<path_to_scanner>/scanner.py", replacing <path_to_scanner> with the directory containing scanner.py and WavePlotImager
